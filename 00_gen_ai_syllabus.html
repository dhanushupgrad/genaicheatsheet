📘 GENERATIVE AI -- EXAM-READY NOTES

1. Introduction to Generative AI

Generative AI refers to models that create new data similar to training data.

Key Points

  • Generates text, images, audio, video, code
  • Learns data distribution, not just labels
  • Uses probabilistic modeling

Generative vs Predictive Models

Generative Predictive
Creates new data Predicts labels
Models full distribution Models conditional probability
Examples: GPT, Diffusion Examples: Logistic Regression

2. Transformer Architecture (Overview)

  • Encoder, Decoder, Encoder--Decoder
  • Attention mechanism
  • Positional encoding

Model Types

  • GPT → Decoder-only → text generation
  • BERT → Encoder-only → understanding
  • T5 → Encoder--Decoder → seq-to-seq tasks

3. Diffusion Models ⭐ (High-Weight Topic)

Used mainly for image generation

Working Principle

  1. Forward Process -- Add noise step-by-step
  2. Reverse Process -- Learn to remove noise

Applications

  • Text-to-image generation
  • Inpainting
  • Style transfer
  • Image enhancement

Examples

Stable Diffusion DALL·E MidJourney

4. Prompt Engineering

Designing prompts to guide model behavior.

Types of Prompting

  • Zero-shot -- No examples
  • Few-shot -- With examples
  • Chain-of-Thought (CoT) -- Step-by-step reasoning

Prompt Design Strategies

  • Clear instructions
  • Role-based prompts
  • Output format specification

5. Multimodal AI

Models that handle multiple data types together

Core Concepts

  • Text + Image + Audio
  • Joint embedding spaces

Examples

  • CLIP -- text--image alignment
  • BLIP / BLIP-2 -- vision-language transformers
  • GPT-4V -- visual + language understanding

6. Evaluation of Generative AI

Text Metrics

  • BLEU -- n-gram overlap
  • ROUGE -- recall-based
  • METEOR -- synonym-aware

Image Metrics

  • FID -- realism & diversity
  • Inception Score -- quality & uniqueness

Human Evaluation

  • Helpfulness
  • Creativity
  • Coherence

8. RAG (Retrieval-Augmented Generation)
