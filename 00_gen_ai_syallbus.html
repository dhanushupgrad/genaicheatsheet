Generative Artificial Intelligence

Exam Syllabus

(A)

Unit I: Introduction to Generative Artificial Intelligence

  • Concept of Generative AI
  • Generative vs Predictive Models
  • Types of Generative Content (Text, Image, Audio, Video, Code)

Unit II: Transformer Architectures

  • Foundation Models
  • Encoder, Decoder, Encoder–Decoder Architectures
  • Attention Mechanism
  • Positional Encoding
  • Applications of Transformer Models
  • GPT (decoder-only): autoregressive, next-token prediction
  • BERT (encoder-only): masked token prediction for contextual embeddings
  • T5 (encoder-decoder): sequence-to-sequence multitask learning

Unit III: Diffusion Models

  • Diffusion Model Workflow
  • Noise Injection and Denoising Process
  • Text-to-Image Models
  • Applications of Diffusion Models
  • Forward process: add noise to data (e.g., images)
  • Reverse process: learn how to denoise step-by-step
  • Stable Diffusion
  • DALL·E, MidJourney

Applications

  • Text-to-image generation
  • Inpainting: filling in masked image regions
  • Style transfer and enhancement

Unit IV: Prompt Engineering

  • Zero-shot Prompting
  • Few-shot Prompting
  • Chain-of-Thought Prompting
  • Prompt Design Strategies
  • Prompt Engineering Tools
  • Zero-shot: "What is the capital of France?"
  • Few-shot: Provide examples for structure
  • Chain-of-Thought: encourage reasoning step-by-step

Unit V: Multimodal Artificial Intelligence

  • Multimodal Learning Concepts
  • Vision–Language Models
  • Text–Image–Audio Integration
  • Applications of Multimodal AI
  • CLIP: connects text and images via embeddings
  • BLIP, BLIP-2: vision-language transformers
  • GPT-4V (Vision): image + language prompts
  • Flamingo, Gemini: text, image, video, interleaved understanding

Unit VI: Evaluation of Generative AI Systems

  • Text Evaluation Metrics
  • Image Evaluation Metrics
  • Human Evaluation Techniques

Text Metrics

  • BLEU: overlap in n-grams (machine translation)
  • ROUGE: recall-based overlap for summarization
  • METEOR: synonym-aware evaluation

Image Metrics

  • FID (Fréchet Inception Distance): visual realism and diversity
  • Inception Score (IS): quality and uniqueness

🧰 Tools & Libraries

  • Hugging Face Transformers, Diffusers
  • OpenAI APIs (ChatGPT, DALL·E)
  • Replicate, Stability AI, DreamStudio
  • PromptLayer, LangChain (prompt chaining)
  • Colab + Gradio for quick interfaces
(B)

Unit VII: Fine-Tuning of Language Models

  • Fine-Tuning Approaches
  • Parameter-Efficient Fine-Tuning (PEFT)
  • LoRA and QLoRA
  • Adapters
  • Fine-Tuning Tools and Frameworks

Application

  • Hugging Face PEFT + Transformers
  • bitsandbytes for quantization
  • LLaMA.cpp, GGML for lightweight deployment

Unit VIII: Advanced Training Techniques

  • Reinforcement Learning from Human Feedback (RLHF)
  • Direct Preference Optimization (DPO)
  • Instruction Tuning
  • Constitutional AI

Unit IX: Retrieval-Augmented Generation (RAG)

  • Need for Retrieval-Augmented Generation
  • RAG Architecture
  • Embedding Models
  • Vector Databases
  • Orchestration Frameworks

Unit X: AI Agents and Multi-Agent Systems

  • Agent-Based AI Systems
  • Agent Frameworks
  • Agent Architectures
  • Multi-Agent Communication
  • Collaborative Agent Systems

Unit XI: Applications of Generative AI

  • Code Generation
  • Scientific Applications
  • Creative Applications
  • Search and Productivity Applications

Unit XII: Ethics, Alignment, and AI Safety

  • Bias and Fairness in AI Models
  • Safety Risks in Generative AI
  • Red Teaming Techniques
  • Transparency and Interpretability
  • Explainable AI

Unit XIII: Tools and Platforms

  • Fine-Tuning and Experimentation Tools
  • Retrieval and Tool-Use Frameworks
  • Agent Development Frameworks
  • Model Monitoring and Tracking Tools
  • APIs and Hosted AI Models

Unit XIV: Project and Practical Applications

  • Mini Projects
  • Capstone Project
  • Deployment and Demonstration
